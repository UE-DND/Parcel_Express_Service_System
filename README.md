# 包裹快递服务系统 (Parcel Express Service System)
> 这个项目没有什么技术含量，仅作为一个标准的Maven项目示范。

## 项目简介

面向对象程序设计课程项目的附加题，模拟FedEx、DHL和UPS等快递公司的包裹投递服务系统并实现不同种类的包裹及其邮费计算规则。
